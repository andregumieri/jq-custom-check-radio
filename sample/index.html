Custom Check & Radio jQuery Plugin Samples

Apply to all radio and radio inputs

HTML code

<form id="form1">
	<div>
		Radio: Group 1<br />
		<input type="radio" name="form1-radio" /> Radio 1
		<input type="radio" name="form1-radio" /> Radio 2
		<input type="radio" name="form1-radio" /> Radio 3
	</div>
	
	<div>
		Radio: Group 2<br />
		<input type="radio" name="form1-radio2" /> Radio 1
		<input type="radio" name="form1-radio2" /> Radio 2
		<input type="radio" name="form1-radio2" /> Radio 3
	</div>
</form>

<div>
	Checkboxes<br />
	<label><input type="checkbox" name="form1-check-1" /> Check 1</label>
	<label><input type="checkbox" checked="checked" name="form1-check-2" /> Check 2</label>
	<label><input type="checkbox" name="form1-check-3" /> Check 3</label>
	<label><input type="checkbox" checked="checked" name="form1-check-4" /> Check 4</label>
	<label><input type="checkbox" name="form1-check-5" /> Check 5</label>
</div>

Javascript code

$("#form1 input[type=radio],#form1 input[type=checkbox]").checkAndRadio();
